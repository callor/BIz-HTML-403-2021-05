Flex 를 이용한 화면 구현

초창기 HTML은 위에서 아래방향으로 진행되는 Layout이 주를 이뤘다

가로방향으로 Content를 배열하는 layout 필요하게 된다

여러가지 Hack(Tip)들이 등장을하고 다양한 방법이 동원된다

표준 HTML 사용법과는 다소 거리가 있어서

여러가지 문제들을 일으키고

원하는대로 Layout이 만들어지지 않았다

대표적인 방법으로 div box를 floatting하여 배열하는 방법이

사용되었다

대표적인 box model

box model tag를 float로 설정을 하면

box width에 따라 화면에서 한라인에

여러 tag를 배열할수 있다

그런 float로 만든 layout은

floating되기를 원하지 않는 tag에 까지

영향을 미치게 된다

inline-block

float을 사용하여 layout을 구성하던

HTML 디자이너 들에게 반가운 소식이 전해진다

HTML5의 표준에 box model을 변경할수 있는 속성이

추가된다

display 속성

기본적인 box model을 변형(변경)하여 적용할수 있속성

block -> inline

inline -> block

div tag에 display를 inline으로 설정했더니

갑자기 block 성질을 버리고 한 라인에

여러개의 box를 배열할수 있게 되었다

그런데 inline 속성은

box의 width를 설정할수 없다.

다행히 inline 속에서 block의 성질을 포함한

inline-block 속성값이 추가된다

드디어 div box를 사용하여

원하는대로 width를 설정하고

한 라인에 여러 박스를 배치할수 있게 되었다
